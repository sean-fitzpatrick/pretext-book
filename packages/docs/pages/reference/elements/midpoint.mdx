# `<midpoint>`

A centered dot used sometimes like a decorative dash.
The unicode character is &#xb7;, which is enteredin HTML as `&#xb7;`.
Used with the self-closing syntax: `<midpoint/>`.

## Syntax

### Attributes

<AttrDisplay name="midpoint" />

### Children

The following can be used as children.

<ChildrenDisplay name="midpoint" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="midpoint" />
