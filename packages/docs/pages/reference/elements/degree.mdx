# `<degree>`

The degree symbol. Useful for both angles and temperatures.
The unicode character is &#xb0;, which is entered in HTML as `&#xb0;`.
Used with the self-closing syntax `<degree/>`.

## Syntax

### Attributes

<AttrDisplay name="degree" />

### Children

The following can be used as children.

<ChildrenDisplay name="degree" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="degree" />

## Examples
```ptx-example
<FRAGMENT parents="p">
    During the <pretext/> workshop, the temperature was 97<degree/>,
    without accounting for humidity.
</FRAGMENTS>
```
