# `<nbsp>`

Inserts a non-breaking space. Entered as `&#xa0;` in HTML.
Always used with the self-closing syntax `<nbsp/>`.

## Syntax

### Attributes

<AttrDisplay name="nbsp" />

### Children

The following can be used as children.

<ChildrenDisplay name="nbsp" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="nbsp" />

## Examples
```ptx-example
<FRAGMENT parents="p">
    When we refer to Dr.<nbsp/>Pepper, we like to ensure their title and name appear on the same line.
</FRAGMENT>
```
