# `<permille>`

Or, per thousand, like a percent sign, but with one more zero.
Always used as a self-closing tag: `<permille/>`

## Syntax

### Attributes

<AttrDisplay name="permille" />

### Children

The following elements can be used as children.

<ChildrenDisplay name="permille" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="permille" />

## Examples
```ptx-example
<FRAGMENT parents="p">
    I always give 1000<permille/>.
</FRAGMENT>
```
