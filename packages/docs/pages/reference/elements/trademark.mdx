# `<trademark>`

Produces the trademark symbol.
The unicode character is &#x2122;, which is entered in HTML as `&#x2122;`.
Used with the self-closing syntax `<trademark/>`

## Syntax

### Attributes

<AttrDisplay name="trademark" />

### Children

The following can be used as children.

<ChildrenDisplay name="trademark" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="trademark" />

## Examples
```ptx-example
<FRAGMENTparents="p">
Whizz-Banger<trademark/> is a trademark of MegaCoro<registered/>.
</FRAGMENT>
```
