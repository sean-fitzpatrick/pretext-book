# `<swungdash>`

A decorative dash, like a tilde, but bigger, and centered.
Always used as a self-closing tag: `<swungdash/>`

## Syntax

### Attributes

<AttrDisplay name="swungdash" />

### Children

The following elements can be used as children.

<ChildrenDisplay name="swungdash" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="swungdash" />
