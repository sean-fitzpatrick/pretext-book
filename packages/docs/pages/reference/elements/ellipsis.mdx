# `<ellipsis>`

Inserts the ellipsis (...). Used as the self-closing tag `<ellipsis/>`.

<AttrDisplay name="ellipsis" />

### Children

<ChildrenDisplay name="ellipsis" />

<ParentsDisplay name="ellipsis" />
