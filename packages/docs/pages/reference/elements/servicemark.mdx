# `<servicemark>`

Produces the servicemark symbol, used to identify a service, as opposed to a product.
The unicode character is &#x2120;, which is entered in HTML as `&#x2120;`.
Used with the self-closing syntax `<servicemark/>`

## Syntax

### Attributes

<AttrDisplay name="servicemark" />

### Children

The following can be used as children.

<ChildrenDisplay name="servicemark" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="servicemark" />
