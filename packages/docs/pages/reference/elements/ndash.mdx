# `<ndash>`

The en-dash character. Longer than a hyphen, but shorter than an em-dash.
The unicode character is &#8211;, which is entered in HTML as `&#8211;`.
Always used with the self-closing syntax `<ndash/>`.

## Syntax

### Attributes

<AttrDisplay name="ndash" />

### Children

The following can be used as children.

<ChildrenDisplay name="ndash" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="ndash" />

## Examples
```ptx-example
<FRAGMENT parents="p">
    Our work week is often Monday<ndash/>Saturday.
</FRAGMENT>
```
