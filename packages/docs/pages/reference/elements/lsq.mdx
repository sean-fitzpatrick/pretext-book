# `<lsq>`

Left single quote. For those occasional times when you want to open a single quote but not close it,
or close it after some intervening content that can't go inside `<sq>`.
Produces the character `&#x2018;` in HTML. Always used with self-closing syntax `<lsq/>`.

## Syntax

### Attributes

<AttrDisplay name="lsq" />

### Children

The following can be used as children.

<ChildrenDisplay name="lsq" />

### Child of
