# `<mdash>`

The em-dash character. Longer than an en-dash.
The unicode character is &#8212;, which is entered in HTML as `&#8212;`.
Always used with the self-closing syntax `<mdash/>`.

## Syntax

### Attributes

<AttrDisplay name="mdash" />

### Children

The following can be used as children.

<ChildrenDisplay name="mdash" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="mdash" />

## Examples
```ptx-example
<FRAGMENT parents="p">
    Some people <mdash/> who may or may not be grammatically correct <mdash/> like using the em-dash for punctuation.
</FRAGMENT>
```
