# `<ca>`

Inserts the abbreviation "ca." for *circa*.
Used exclusively as the self-closing tag `<ca/>`,
and should work anywhere you would write text.

## Syntax

### Attributes

<AttrDisplay name="ca" />

### Children

The following elements can be used as children.

<ChildrenDisplay name="ca" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="ca" />

## Examples
```ptx-example
<FRAGMENT parents="p">
    Rock music is generally recognized to have peaked <ca/> 1996.
</FRAGMENT>
```
