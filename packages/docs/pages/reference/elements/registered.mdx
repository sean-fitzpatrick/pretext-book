# `<registered>`

The registered trademark symbol.
The unicode symbol is &#xae;, which is entered as `&#xae;` in HTML.
Used with the self-closing syntax `<registered/>`.

## Syntax

### Attributes

<AttrDisplay name="registered" />

### Children

The following can be used as children.

<ChildrenDisplay name="registered" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="registered" />

## Examples
```ptx-example
<FRAGMENT parents="p">
    Registered trademark of MegaCorp<registered/>.
</FRAGMENT>
```
