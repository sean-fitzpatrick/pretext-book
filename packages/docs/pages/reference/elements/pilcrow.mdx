# `<pilcrow>`

Character often used to mark the start of a paragraph.
The unicode character is &#xb6;, which is entered in HTML as `&#xb6;`.
Always used with the self-closing syntax `<pilcrow/>`.

## Syntax

### Attributes

<AttrDisplay name="pilcrow" />

### Children

The following can be used as children.

<ChildrenDisplay name="pilcrow" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="pilcrow" />

## Examples
```ptx-example
<FRAGMENT parents="p">
    <pilcrow/>A pilcrow is at the beginning of this sentence.
</FRAGMENT>
```
