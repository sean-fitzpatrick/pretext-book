# `<copyright>`

Produces the copyright symbol.
The unicode character is &#xa9;, which is entered as `&#xa9;` in HTML.
Used with the self-closing syntax `<copyright/>`.

## Syntax

### Attributes

<AttrDisplay name="copyright" />

### Children

The following can be used as children.

<ChildrenDisplay name="copyright" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="copyright" />

## Examples
```ptx-eample
<FRAGMENT parents="p">
  Copyright <copyright/> 1993, all rights reserved.
</FRAGMENT>
```
