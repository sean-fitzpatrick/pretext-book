# `<rsq>`

Right single quote. For those occasional times when you want to close a single quote that wasn't opened,
or close it after some intervening content that can't go inside `<sq>`.
Produces the character `&#x2019;` in HTML. Always used with self-closing syntax `<rsq/>`.

## Syntax

### Attributes

<AttrDisplay name="rsq" />

### Children

The following elements can be used as children.

<ChildrenDisplay name="rsq" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="rsq" />
