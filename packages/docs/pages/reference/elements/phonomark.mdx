# `<phonomark>`

The phonomark character, which is like regsistered trademark, but with a P instead of an R.
Used for music.
The unicode character is &#x2117;, which is entered in HTML as `&#x2117;`.
Used with the self-closing syntax `<phonomark/>`.

## Syntax

### Attributes

<AttrDisplay name="phonomark" />

### Children

The following can be used as children.

<ChildrenDisplay name="phonomark" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="phonomark" />
